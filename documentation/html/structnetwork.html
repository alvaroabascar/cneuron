matrix_double* network::biases

Array of matrices for the biases. Each matrix will actually contain a single column, with biases[i][k] containing the bias of the kth neuron in layer "i".

int network::n_layers

Number of layers of the network, including input and output layers.

int* network::net_structure

Array indicating the number of neurons in each layer. Eg. for a net with 1, 20 and 10 neurons, net_structure = {1, 20, 10}

matrix_double* network::weights

Array of matrices for the weights. weights[i][j][k] will contain the weight which links neuron "k" in layer "i" with neuron "j" in layer "i+1"
